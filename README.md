## Quick Start

To start Jupyter Lab, simply run:

```bash
uv run jupyter lab
```

This command will automatically:
1.  Create a virtual environment (if one doesn't exist).
2.  Install/Sync all dependencies.
3.  Launch the Jupyter Lab interface in your browser.
